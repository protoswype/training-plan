# 16W Makrozyklus – Sportklettern

Personal climbing training plan app. Accessible from anywhere via GitHub Pages.

## GitHub Pages Setup

1. Push this repo to GitHub (private repo is fine — Pages works on private repos with a Pro/Team account, or make it public).

2. Go to **Settings → Pages**.

3. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`

4. Click **Save**.

5. GitHub will publish the site. URL:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```
   Shown in Settings → Pages once deployed (takes ~1 min on first push).

## No build required

Static HTML/CSS/JS. Push changes to `main` → Pages redeploys automatically within ~1 minute.
